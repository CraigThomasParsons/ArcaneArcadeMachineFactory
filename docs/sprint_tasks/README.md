# Sprint Tasks

This folder contains executable sprint task backlogs for the game-factory MVP.

Each sprint includes:

1. 0-tasks.md: scope lock and success metrics.
2. 1-tasks.md: architecture and interface baseline.
3. 2-tasks.md: design refinement and contracts.
4. 3-tasks.md: implementation slice.
5. 4-tasks.md: validation and hardening.
6. 5-tasks.md: integration completion and handoff.
